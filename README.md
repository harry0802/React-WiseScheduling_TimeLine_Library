# WiseScheduling Timeline Library

React 前端專案 - 生產排程時間軸元件庫

## 系統需求

- Node.js 16+
- npm 或 yarn

## 安裝與啟動

複製原始碼到本地：

```bash
git clone git@gitlab.com:ts_team/flaskplastic.git
cd flaskplastic/frontend
```

安裝依賴：

```bash
npm install
```

啟動開發伺服器：

```bash
npm start
```

## 專案結構

```text
frontend/
├── src/              # 原始碼
├── public/           # 靜態資源
└── package.json      # 專案設定
```
